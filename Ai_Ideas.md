AI Use Cases for This System
                                                                                                                                                                
  1. Anomaly Detection
  Flag unusual usage — e.g. someone active at 3 AM, sudden usage drop for a team, or a machine logging activity with no user logged in.

  2. License Optimization Advisor
  AI predicts how many licenses you actually need per department based on usage patterns — reducing waste and identifying over-allocated seats.

  3. Auto Report Summarization
  Append a Claude-generated paragraph to every PDF report: "Architecture team usage dropped 18% this month, driven by 3 idle machines..." — plain English
  insight, not just charts.

  4. Natural Language Query
  A chat box: "Who used Revit the most in March?" or "Which department had the most ghost machines last quarter?" — no need to navigate filters.

  5. Usage Forecasting
  Predict next month's peak usage days using historical trends — helps IT plan maintenance windows.

  6. Smart Idle Prediction
  AI learns each user's typical rhythm and flags when someone is heading toward idle/ghost status before it happens.

  7. Department Efficiency Scoring
  AI-derived score beyond raw hours — combining active days, app variety, idle ratio — to give a fairer productivity picture.
